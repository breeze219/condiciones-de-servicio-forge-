# Política de Privacidad - Forgefree

Forgefree es un bot de Discord que gestiona mensajes de bienvenida personalizados. A continuación, explicamos qué información recopilamos y cómo la usamos.

## Datos recopilados
- **IDs de usuarios**: para identificar cuentas.
- **Mensajes de bienvenida**: configurados por los usuarios para personalizar saludos.

## Uso de datos
- Los datos se utilizan exclusivamente para mejorar las respuestas automáticas del bot.
- La información se almacena en servidores seguros y no se comparte con terceros.
